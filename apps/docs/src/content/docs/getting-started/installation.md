---
title: Installation
description: Install Minions Oss-features packages
---

## TypeScript / Node.js

```bash
npm install @minions-oss-features/sdk minions-sdk
```

## Python

```bash
pip install minions-oss-features
```

## CLI

```bash
npm install -g @minions-oss-features/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
