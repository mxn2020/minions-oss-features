# Minions Oss-features

Feature backlog, daily picks, implementation plans, and acceptance criteria

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-oss-features/sdk`) — Core library for Node.js
- **Python SDK** (`minions-oss-features`) — Python bindings
- **CLI Tool** (`@minions-oss-features/cli`) — Command-line interface
- **Web App** — Interactive playground at [oss-features.minions.wtf](https://oss-features.minions.wtf)
